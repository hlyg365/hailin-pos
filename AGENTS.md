# 海邻到家 V6.0 统一跨端架构

## 项目概览

海邻到家是一个面向连锁便利店的智能收银与营销管理系统，采用 **单应用 SPA 架构**，所有功能模块通过路由访问。V6.0 版本实现了完整的"总部强管控、门店轻运营"架构。

### 企业标识

- **Logo 文件**：`/public/logo.png`
- **设计风格**：蓝绿撞色设计，房屋轮廓 + HLDJ 缩写，中英文品牌名称
- **应用范围**：所有页面顶部导航、侧边栏、登录页、设置页等

### 六端入口

| 终端 | 访问路径 | 功能定位 |
|------|----------|----------|
| 首页 | `/` | 四端入口、快捷功能、今日概览 |
| 收银台 | `/pos/cashier` | AI智慧收银、聚合支付、离线支持 |
| 管理后台 | `/dashboard` | 总部管理、数据分析、供应链、财务 |
| 小程序商城 | `/mini` | 商品浏览、团购接龙、线上销售 |
| 店长助手 | `/assistant` | 移动端管理、库存盘点、要货缴款 |
| 会员管理 | `/pos/member` | 会员识别、积分管理 |

## V6.0 核心功能

### 1. AI 智慧收银 (收银台)
- **AI 条码识别**：扫描未知条码，自动联网查询商品库，智能回填
- **AI 视觉识别**：非标品（水果/烘焙）摄像头识别，自动估算重量
- **聚合支付**：微信、支付宝、云闪付、现金、数字人民币、会员卡
- **离线收银**：断网正常收银，网络恢复自动同步
- **晚8点清货**：20:00-23:00 自动8折
- **负库存检查**：严禁负库存销售

### 1.1 收银设备支持
- **客显屏**：独立窗口显示商品、金额、支付状态，支持欢迎/待付/已付三种模式
- **小票打印机**：ESC/POS 协议，支持 58mm/80mm 纸宽，自动打印小票
- **标签打印机**：价格标签批量打印，支持条码
- **钱箱**：自动弹出提示，现金支付自动触发
- **电子秤**：实时重量读取，支持去皮、校准
- **扫码枪/摄像头**：支持 USB 扫码枪和摄像头 AI 扫码

### 2. 总部管理后台
- **工作台**：销售统计、订单概览、待办提醒
- **门店管理**：多门店数据查看、状态监控
- **供应链**：采购订单、智能配送调度、门店收货、盘点管理
- **财务管理**：资金归集、缴款单、对账中心、加盟结算
- **会员管理**：CRM、等级权益、标签营销
- **人员管理**：员工账号、考勤记录
- **促销管理**：满减、折扣、买赠、时段促销、会员专享
- **价格体系**：统一定价、区域定价、门店特价、时段价格

### 3. BI 数据分析中心
- **实时大屏**：当日营收、客单价、订单数实时跳动
- **时段分析**：识别高峰低谷，优化排班
- **ABC分析**：识别畅销品与滞销品
- **毛利分析**：单品毛利、分类毛利、门店毛利对比
- **预警中心**：库存积压、临期商品、异常退单预警
- **门店排名**：营收、订单、毛利多维度排行

### 4. 门店运营中心
- **门店档案**：基本信息、营业时间、区域管理
- **调拨管理**：门店间商品调拨审批流程
- **巡店管理**：督导巡店、评分、整改跟踪
- **电子工单**：通知公告、培训资料、设备维修

### 5. 权限管理
- **角色定义**：超级管理员、区域督导、门店店长、收银员
- **权限配置**：按模块、按操作细粒度授权
- **操作日志**：用户操作可追溯

### 6. 小程序商城
- **商品浏览**：分类筛选、搜索、购物车
- **社区团购**：团长发起、实时接龙、门店核销
- **外卖对接**：美团/饿了么聚合接单
- **会员中心**：积分、余额、优惠券

### 7. 店长助手
- **首页仪表盘**：今日统计、待办提醒、库存预警
- **库存管理**：实时库存、要货申请、盘点
- **缴款功能**：现金收款缴款、银行转账
- **数据报表**：销售趋势、热销排行

## 特殊功能

### 晚8点清货模式
- **触发时间**：20:00 - 23:00
- **优惠内容**：全场商品 8 折
- **显示标识**：顶部显示"清货模式 8折"标签

### 离线支持
- **检测机制**：监听网络状态，离线时显示提示

### 负库存检查
- 严禁负库存销售，出库时严格校验库存数量

## 数据模型

### 核心类型
- `Store` - 门店信息
- `Employee` - 员工信息
- `Product` - 商品信息（含标品/非标品标识）
- `Inventory` - 库存信息
- `Order` - 订单信息（支持POS/小程序/外卖/团购）
- `Member` - 会员信息
- `RestockRequest` - 要货申请
- `TransferRequest` - 调拨申请
- `Inspection` - 巡店记录
- `WorkOrder` - 电子工单
- `DayStatement` - 日结单
- `Role` - 角色权限

## 开发指南

### 持久化存储
系统使用 Zustand + localStorage 实现数据持久化：
- `hailin-settings`: 系统设置（支付、促销、硬件等）
- `hailin-miniprogram`: 小程序设置（Banner、分类等）

### 安装依赖
```bash
cd apps/pos-app
pnpm install
```

### 开发模式
```bash
cd apps/pos-app
pnpm dev
```

### 构建
```bash
cd apps/pos-app
pnpm build
```
