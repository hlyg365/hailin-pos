# 海邻到家 V2.0 统一跨端架构

## 项目概览

海邻到家是一个面向社区便利店的智能收银与营销管理系统，采用 **单应用 SPA 架构**，所有功能模块通过路由访问。

### 四端入口

| 终端 | 访问路径 | 功能定位 |
|------|----------|----------|
| 首页 | `/` | 四端入口、快捷功能 |
| 收银台 | `/pos/cashier` | 门店快速收银、硬件集成 |
| 小程序商城 | `/mini` | 商品浏览、团购接龙、线上销售 |
| 总部管理后台 | `/dashboard` | 多门店管理、数据分析 |
| 店长助手 | `/assistant` | 移动端管理、库存盘点 |

## 功能详情

### 首页 (`/`)
- 四端入口卡片：收银台、管理后台、小程序商城、店长助手
- 快捷功能入口：快速收银、会员管理、商品管理、库存盘点
- 系统状态显示

### 收银台 (`/pos/cashier`)
- **商品浏览**：分类筛选、搜索、扫码添加
- **购物车**：数量修改、删除、清空
- **晚8点清货**：20:00-23:00 自动8折
- **多种支付**：现金、微信、支付宝、会员卡
- **会员识别**：扫码/手机号识别会员
- **挂单功能**：临时挂起订单，稍后继续

### 管理后台 (`/dashboard`)
- **工作台**：销售统计、订单概览、门店数据
- **门店管理**：多门店数据查看
- **商品管理**：商品列表、价格管理
- **订单管理**：订单查询、退款处理
- **会员管理**：会员等级、积分管理

### 小程序商城 (`/mini`)
- **首页**：商品列表、分类筛选
- **购物车**：添加、修改、结算
- **我的**：会员中心、订单管理
- **团购接龙**：团长发起、实时参与

### 店长助手 (`/assistant`)
- **首页仪表盘**：今日统计、待办提醒、库存预警
- **快捷功能**：库存、盘点、采购、报表
- **数据可视化**：销售趋势图表

- **触发时间**：20:00 - 23:00
- **优惠内容**：全场商品 8 折

- **检测机制**：监听网络状态，离线时显示提示
