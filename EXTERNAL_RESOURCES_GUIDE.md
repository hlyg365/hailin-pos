# 外部资源执行步骤指南

## 🚀 快速开始

按照以下步骤，准备好所有必要的外部资源，预计总时间：**1-2周**

---

## 第1步：微信小程序注册（1-2天）

### 1.1 注册小程序

**所需材料**：
- 营业执照
- 法人身份证
- 对公银行账户

**执行步骤**：

1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册" > 选择"小程序"
3. 填写账号信息：
   - 账号名称：海邻到家
   - 账号介绍：社区便利店智能收银系统
4. 主体认证：
   - 上传营业执照
   - 填写企业信息
   - 填写对公账户
5. 管理员信息：
   - 填写管理员姓名、身份证、手机、邮箱
6. 提交审核

**获得**：小程序账号、AppID

**时间**：1-7个工作日

---

### 1.2 小程序认证（1-3天）

**所需材料**：
- 营业执照
- 法人身份证
- 认证费 300元

**执行步骤**：

1. 登录微信公众平台
2. 进入"设置" > "基本设置" > "微信认证"
3. 选择"企业认证"
4. 填写企业信息
5. 支付认证费 300元
6. 提交审核

**获得**：微信支付权限、模板消息权限

**时间**：1-3个工作日

---

## 第2步：微信支付开通（2-5天）

### 2.1 注册商户号（1-3天）

**所需材料**：
- 营业执照
- 法人身份证
- 对公银行账户

**执行步骤**：

1. 访问 https://pay.weixin.qq.com/
2. 点击"接入支付"
3. 选择"企业"
4. 填写商户信息：
   - 商户名称：与营业执照一致
   - 商户简称：海邻到家
5. 上传资质材料
6. 提交审核

**获得**：商户号

**时间**：1-5个工作日

---

### 2.2 配置支付（1天）

**执行步骤**：

1. 登录微信支付商户平台
2. 进入"账户中心" > "API安全"
3. 设置API密钥（32位字符串）
4. **保存密钥**（仅显示一次）
5. 下载API证书
6. 进入"产品中心" > "开发配置"
7. 配置回调地址：
   - 支付回调：`https://api.hailin.com/api/pay/wechat/notify`
   - 退款回调：`https://api.hailin.com/api/pay/wechat/refund/notify`
8. 进入"产品中心" > "AppID授权"
9. 关联小程序（输入小程序AppID）

**获得**：API密钥、API证书

**时间**：1个工作日

**费用**：交易费率0.6%

---

## 第3步：域名和服务器（3-10天）

### 3.1 购买域名（即时）

**执行步骤**：

1. 访问阿里云/腾讯云域名注册
2. 搜索可用域名（如：hailin.com）
3. 购买域名（50-100元/年）
4. 完成支付

**获得**：域名

**时间**：即时

**费用**：50-200元/年

---

### 3.2 域名解析（10分钟-48小时）

**执行步骤**：

1. 登录域名管理后台
2. 添加DNS记录：
   - A记录：`@` → 服务器IP
   - A记录：`api` → 服务器IP
3. 保存设置
4. 等待DNS生效

**测试**：`ping api.hailin.com`

**时间**：10分钟-48小时

---

### 3.3 配置HTTPS（10分钟-24小时）

**执行步骤**：

1. 申请免费SSL证书（Let's Encrypt或云服务商提供）
2. 提交域名信息
3. 验证域名所有权
4. 下载证书文件（.crt 和 .key）
5. 配置Nginx：
   ```nginx
   server {
       listen 443 ssl;
       server_name api.hailin.com;
       ssl_certificate /path/to/cert.crt;
       ssl_certificate_key /path/to/cert.key;
   }
   ```
6. 重启Nginx

**测试**：访问 https://api.hailin.com

**时间**：10分钟-24小时

**费用**：免费

---

### 3.4 购买服务器（即时-1小时）

**执行步骤**：

1. 访问阿里云/腾讯云
2. 选择服务器配置：
   - CPU：2核
   - 内存：4GB
   - 硬盘：40GB SSD
   - 带宽：5Mbps
3. 选择操作系统：Ubuntu 20.04
4. 购买服务器（1000-5000元/年）
5. 重置密码
6. 配置安全组（开放80、443、5000端口）

**获得**：服务器IP地址

**时间**：即时（购买），1小时（配置）

**费用**：1000-5000元/年

---

### 3.5 安装环境（1-2小时）

**执行步骤**：

1. 连接服务器
2. 安装Node.js 18+：
   ```bash
   curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
   sudo apt-get install -y nodejs
   ```

3. 安装PostgreSQL：
   ```bash
   sudo apt install postgresql postgresql-contrib
   ```

4. 安装Nginx：
   ```bash
   sudo apt install nginx
   ```

5. 安装PM2：
   ```bash
   sudo npm install -g pm2
   ```

**获得**：可用的服务器环境

**时间**：1-2小时

---

## 第4步：配置后端服务（2-3小时）

### 4.1 创建数据库

**执行步骤**：

1. 连接到PostgreSQL：
   ```bash
   sudo -u postgres psql
   ```

2. 创建数据库和用户：
   ```sql
   CREATE DATABASE hailin;
   CREATE USER hailin_user WITH PASSWORD 'your_password';
   GRANT ALL PRIVILEGES ON DATABASE hailin TO hailin_user;
   \q
   ```

**获得**：数据库连接信息

**时间**：10分钟

---

### 4.2 部署后端代码

**执行步骤**：

1. 克隆项目代码
2. 安装依赖：
   ```bash
   pnpm install
   ```
3. 构建项目：
   ```bash
   pnpm build
   ```
4. 创建环境变量文件 `.env`：
   ```env
   DATABASE_URL=postgresql://hailin_user:password@localhost:5432/hailin
   BASE_URL=https://api.hailin.com
   PORT=5000
   WECHAT_APP_ID=your_app_id
   WECHAT_MCH_ID=your_mch_id
   WECHAT_MCH_KEY=your_mch_key
   ```
5. 启动服务：
   ```bash
   pm2 start npm --name "hailin-api" -- start
   pm2 save
   ```

**获得**：运行中的后端服务

**时间**：1小时

---

## 第5步：配置小程序（1小时）

### 5.1 配置服务器域名

**执行步骤**：

1. 登录微信公众平台
2. 进入"开发" > "开发管理" > "开发设置"
3. 配置服务器域名：
   - request合法域名：`https://api.hailin.com`
   - uploadFile合法域名：`https://api.hailin.com`
   - downloadFile合法域名：`https://api.hailin.com`
   - socket合法域名：`wss://api.hailin.com`

**时间**：5分钟

---

### 5.2 配置隐私协议

**执行步骤**：

1. 在小程序中创建隐私协议页面
2. 进入"设置" > "基本设置" > "隐私指引"
3. 填写隐私保护指引
4. 上传隐私协议文件

**时间**：30分钟

---

### 5.3 配置用户协议

**执行步骤**：

1. 在小程序中创建用户协议页面
2. 在登录页添加用户协议勾选
3. 确保勾选后才能登录

**时间**：30分钟

---

## 第6步：构建收银台APK（1-2小时）

### 6.1 构建APK

**执行步骤**：

1. 安装Java JDK 11+
2. 安装Android SDK
3. 运行构建脚本：
   ```bash
   cd /workspace/projects
   bash scripts/pos-app-quick-build.sh
   ```
4. APK文件位置：
   ```
   android/app/build/outputs/apk/debug/app-debug.apk
   ```

**获得**：可安装的APK文件

**时间**：1-2小时

---

### 6.2 测试APK

**执行步骤**：

1. 连接Android设备（USB调试）
2. 安装APK：
   ```bash
   adb install android/app/build/outputs/apk/debug/app-debug.apk
   ```
3. 启动应用测试所有功能
4. 修复发现的问题

**时间**：1小时

---

## 第7步：提交小程序审核（1-2天）

### 7.1 准备审核材料

**所需材料**：
- 小程序图标（1024x1024px）
- 小程序截图（5-10张，1024x1024px）
- 小程序简介（120字以内）
- 小程序介绍（300字以内）
- 隐私协议
- 用户协议

**执行步骤**：

1. 准备所有材料
2. 检查是否符合规范
3. 整理好所有文件

**时间**：2小时

---

### 7.2 提交审核

**执行步骤**：

1. 登录微信公众平台
2. 进入"版本管理"
3. 点击"上传"
4. 填写版本信息
5. 上传代码
6. 填写审核信息
7. 提交审核

**时间**：30分钟

**审核时间**：1-7个工作日
