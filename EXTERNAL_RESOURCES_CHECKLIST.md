# 外部资源需求清单

## 📋 总览

为了完成三端应用的上线，需要准备以下外部资源和服务。

### 优先级分类
- **P0（必须）**: 不完成无法上线
- **P1（重要）**: 不完成影响用户体验
- **P2（可选）**: 完成后可增强功能

## 📱 1. 微信小程序（P0 - 必须）

### 1.1 小程序注册

**所需材料**：
- [ ] 企业营业执照
- [ ] 法人身份证
- [ ] 对公银行账户信息
- [ ] 企业公章

**执行步骤**：

1. **访问微信公众平台**
   - 网址：https://mp.weixin.qq.com/
   - 点击"立即注册"
   - 选择"小程序"

2. **填写基本信息**
   - 账号名称：海邻到家（建议使用企业名称）
   - 账号介绍：社区便利店智能收银系统
   - 服务类目：商家自营 > 商家自营（百货/超市/便利店）
   - 运营地区：选择实际运营地区

3. **主体认证**
   - 上传营业执照
   - 填写企业信息
   - 上传法人身份证
   - 填写对公账户信息

4. **管理员信息**
   - 管理员姓名
   - 管理员身份证
   - 管理员手机号
   - 管理员邮箱

5. **提交审核**
   - 检查所有信息
   - 提交审核
   - 等待审核通过（1-7个工作日）

**预计时间**：1-7个工作日

**费用**：免费（企业认证）

### 1.2 小程序认证

**所需材料**：
- [ ] 企业营业执照
- [ ] 法人身份证
- [ ] 对公银行账户信息

**预计时间**：1-3个工作日

**费用**：300元/年

**认证后获得**：

### 1.3 小程序信息完善

**执行步骤**：

1. **设置小程序名称**
   - 进入"设置" > "基本设置"

2. **配置服务类目**

### 2.2 微信支付开通

**执行步骤**：

1. **签约**
   - 审核通过后，进行签约
   - 确认费率（一般为0.6%）
   - 确认结算周期（一般为T+1）

2. **获取商户号**
   - 进入微信支付商户平台
   - 复制"商户号"（以1开头）
   - 保存到项目配置中

3. **配置支付密钥**
   - 进入"账户中心" > "API安全"
   - 设置API密钥
   - 复制并保存到项目配置中
   - **重要**：密钥一旦设置，无法查看，只能重置

4. **下载支付证书**
   - 进入"账户中心" > "API安全"
   - 下载"API证书"
   - 解压并保存到项目安全位置
   - **重要**：证书妥善保管，不要泄露

5. **配置回调地址**
   - 进入"产品中心" > "开发配置"
   - 配置支付结果通知URL：
     ```
     https://api.hailin.com/api/pay/wechat/notify
     ```
   - 配置退款结果通知URL：
     ```
     https://api.hailin.com/api/pay/wechat/refund/notify
     ```

**预计时间**：1个工作日

**费率**：一般为0.6%（具体以签约为准）

### 4.2 域名解析

**执行步骤**：

1. **添加DNS记录**
   - 登录域名管理后台
   - 添加A记录：
     - 主机记录：`@` 或 `api`
     - 记录类型：`A`
     - 记录值：服务器IP地址
   - 添加CNAME记录（如果使用CDN）：
     - 主机记录：`www`
     - 记录类型：`CNAME`
     - 记录值：CDN地址

2. **等待DNS生效**
   - DNS生效时间：10分钟-48小时
   - 使用 `ping` 命令测试：
     ```bash
     ping api.hailin.com
     ```

**预计时间**：10分钟-48小时

**费用**：免费

---

### 4.3 HTTPS证书

**执行步骤**：

1. **购买或申请证书**
   - 推荐使用免费证书：Let's Encrypt
   - 或使用云服务商提供的免费证书：
     - 阿里云：免费SSL证书
     - 腾讯云：免费SSL证书

2. **申请证书**
   - 提交域名信息
   - 验证域名所有权（DNS验证或文件验证）
   - 等待证书颁发

3. **下载证书**
   - 下载证书文件（.crt 和 .key）
   - 保存到服务器指定位置

4. **配置证书**
   - Nginx配置示例：
     ```nginx
     server {
         listen 443 ssl;
         server_name api.hailin.com;

         ssl_certificate /path/to/cert.crt;
         ssl_certificate_key /path/to/cert.key;

         location / {
             proxy_pass http://localhost:5000;
         }
     }
     ```

**预计时间**：10分钟-24小时

**费用**：免费（Let's Encrypt）或 0-5000元/年（付费证书）

## 🗄️ 5. 数据库（P0 - 必须）

### 5.1 PostgreSQL配置

**执行步骤**：

1. **安装PostgreSQL**
   ```bash
   # Ubuntu
   sudo apt update
   sudo apt install postgresql postgresql-contrib

   # CentOS
   sudo yum install postgresql postgresql-server
   ```

2. **创建数据库**
   ```bash
   # 切换到postgres用户
   sudo -u postgres psql

   # 创建数据库
   CREATE DATABASE hailin;

   # 创建用户
   CREATE USER hailin_user WITH PASSWORD 'your_password';

   # 授权
   GRANT ALL PRIVILEGES ON DATABASE hailin TO hailin_user;

   # 退出
   \q
   ```

3. **初始化数据库**
   - 运行数据库迁移脚本
   - 初始化基础数据

**预计时间**：30分钟

**费用**：包含在服务器费用中
