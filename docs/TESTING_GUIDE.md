| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 商品扫码 | 扫描商品条码 | 商品添加到购物车 |

| 清空购物车 | 点击清空按钮 | 购物车全部清空 |

```bash

pnpm dev

# 2. 访问预览页
http://localhost:5000/preview

# 3. 手机扫码访问
扫描二维码即可在手机上体验
```

```bash

pnpm run build:apk

```

## 八、常见问题

### Q1: 没有扫码枪怎么测试？

**方案**：
1. 手动输入商品条码（搜索框）
2. 直接点击商品卡片添加
3. 使用手机摄像头扫码（需真机）

### Q2: 没有小票打印机怎么测试？

**方案**：
1. 支付成功后查看订单详情
2. 使用虚拟打印功能（模拟打印）

### Q3: 如何测试离线功能？

**步骤**：
1. 正常登录并加载商品
2. 断开设备网络（飞行模式）
3. 继续收银操作
4. 恢复网络观察同步

### Q4: 多台收银机如何测试？

**方案**：
1. 同一服务器部署
2. 各收银机配置相同服务器地址
3. 使用不同员工账号登录
4. 数据自动同步
