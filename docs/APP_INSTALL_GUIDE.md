# 收银台APP下载安装指南

## 一、获取APP的方式

### 方式一：下载已构建的APK（推荐）

如果您收到了开发团队提供的APK文件：

```
hailin-pos-cashier-v1.0.0.apk
```

直接跳到 **第三步：安装APP**。

---

### 方式二：自行构建APK

#### 1. 环境准备

```bash
# 需要安装的软件
- Node.js 18+
- pnpm 9.0+
- Java JDK 17
- Android SDK
```

#### 2. 克隆项目并构建

```bash
# 克隆代码
git clone <项目地址>
cd hailin-pos-cashier

# 安装依赖
pnpm install

# 构建APK（Debug版本，用于测试）
pnpm run build:apk

# 或构建正式版本
pnpm run build:apk:release
```

#### 3. APK输出位置

```
android/app/build/outputs/apk/
├── debug/app-debug.apk        # 测试版（约50MB）
└── release/app-release.apk    # 正式版（约20MB）
```

---

## 二、传输APK到收银机

### 方法一：U盘传输

```
1. 将 app-debug.apk 复制到U盘
2. U盘插入收银机
3. 在收银机文件管理器中找到APK文件
```

### 方法二：网络下载

```
1. 将APK上传到服务器/网盘
2. 收银机浏览器访问下载链接
3. 下载完成后点击安装
```

### 方法三：ADB安装（需连接电脑）

```bash
# 收银机开启USB调试
设置 → 开发者选项 → USB调试（开启）

# 电脑连接收银机后执行
adb install app-debug.apk

# 或指定设备安装
adb -s <设备ID> install app-debug.apk
```

### 方法四：扫码下载

```
1. 将APK上传到服务器
2. 生成下载二维码
3. 收银机扫码下载
```

---

## 三、安装APP

### Android 安装步骤

```
步骤1: 允许安装未知来源应用
─────────────────────────────
设置 → 安全 → 允许安装未知来源应用（开启）

步骤2: 点击APK文件安装
─────────────────────────────
文件管理器 → 找到APK → 点击安装

步骤3: 授予权限
─────────────────────────────
安装完成后打开APP，授予：
- 相机权限（扫码/AI识别）
- 存储权限（离线数据）
- 网络权限
```

## 四、首次配置

### 1. 登录账号

```
打开APP → 进入登录页面

员工账号：
┌──────────────────────────────┐
│  手机号/工号: admin          │
│  密码: admin123              │
│                              │
│  [登录]                      │
└──────────────────────────────┘
```

### 2. 配置服务器地址（重要！）

```
APP首页 → 左侧导航 → 设置 → 服务器配置

开发测试环境：
http://192.168.1.100:5000  （替换为你的服务器IP）

生产环境：
https://pos.hailin-pos.com

点击 [测试连接] 确认可用后 [保存配置]
```

### 3. 等待数据同步

```
首次登录会自动：
- 下载商品数据到本地
- 同步店铺信息
- 缓存会员数据

等待提示"数据同步完成"即可开始收银
```

## 六、常见问题

### Q1: 安装时提示"解析包错误"？

```
原因：APK文件损坏或Android版本不兼容

解决：
1. 重新下载APK文件
2. 确认Android版本 >= 5.0
3. 检查存储空间是否充足
```

### Q2: 安装时提示"未安装应用"？

```
原因：未开启未知来源安装权限

解决：
设置 → 安全 → 允许安装未知来源应用 → 开启
```

### Q3: APP打开后无法连接服务器？

```
检查项：
1. 收银机是否联网
2. 服务器地址是否正确
3. 服务器是否启动

配置正确地址：
APP设置 → 服务器配置 → 输入正确地址 → 保存
```

### Q4: 如何更新APP版本？

```
1. 下载新版APK
2. 直接安装覆盖（数据会保留）
3. 或卸载旧版后安装新版（需重新登录）
```
