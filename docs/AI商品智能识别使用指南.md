# AI商品智能识别功能使用指南

## 功能概述

AI商品智能识别功能是海邻收银台APP的一项核心创新功能，通过计算机视觉与深度学习技术，实现商品自动识别、称重、计价一体化。

## 技术原理

### 工作流程

```
图像采集 → 特征提取 → 比对匹配 → 数据融合 → 计价输出
    ↓           ↓           ↓           ↓
摄像头拍摄   AI模型分析   商品库匹配   称重传感器
```

### 四个步骤

1. **图像采集**：高清摄像头（200万像素宽动态）捕捉商品图像
2. **特征提取**：深度学习模型（CNN）提取商品关键特征
3. **比对匹配**：与商品特征数据库实时比对
4. **数据融合**：识别结果与称重数据融合，计算总价

整个过程可在0.1-1秒内完成。

## 配置步骤

### 第一步：启用AI识别

1. 打开收银台APP
2. 进入 **硬件管理** → **AI识别**
3. 开启 **启用AI识别** 开关

### 第二步：选择AI品牌

支持以下品牌：
- **爱宝**：国产主流品牌，性价比高
- **由由**：专注生鲜识别
- **食方**：智能称重识别
- **自定义API**：对接自有AI服务

### 第三步：配置触发参数

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 触发重量 | 商品重量达到此值时触发识别 | 20克 |
| 触发模式 | 立即触发/稳定后触发 | 稳定后触发（推荐）|
| 相似度阈值 | 匹配度低于此值不返回结果 | 60% |

### 第四步：商品学习（关键步骤）

每个需要AI识别的商品都需要进行学习：

#### 学习要点

1. **多角度拍摄**：正、反、侧、顶、其他
2. **多种状态**：
   - 带包装/不带包装
   - 装袋/不装袋
   - 整个/切开的
   - 不同份量
3. **样本数量**：每个商品建议学习**20张**照片

#### 学习流程

1. 进入商品管理界面
2. 选择需要学习的商品
3. 点击 **AI学习** 按钮
4. 选择拍摄角度
5. 拍摄商品照片
6. 重复步骤4-5，直到完成20张样本

## 使用方法

### 自动识别流程

1. 将商品放在秤盘上
2. 等待重量稳定（约0.5秒）
3. 系统自动触发AI识别
4. 显示识别结果（商品名称 + 置信度）
5. 确认或手动选择正确商品

### 错误纠正

如果识别错误：

1. 点击识别结果中的 **备选项** 查看
2. 或手动从商品列表选择正确商品
3. 系统会自动学习，提高后续识别准确率

### 手动识别

也可以手动触发识别：

1. 在收银界面点击 **AI识别** 按钮
2. 对准商品拍照
3. 查看识别结果

## 准确率提升技巧

### 1. 光线条件
- 避免强光直射
- 避免阴影遮挡
- 保持光线均匀

### 2. 拍摄角度
- 正面：商品标签/包装正面
- 侧面：商品轮廓特征
- 顶部：俯视角度

### 3. 商品状态
- 覆盖实际销售的各种状态
- 例如：苹果需拍摄带网格包装和散装的

### 4. 持续学习
- 遇到识别错误立即纠正
- 新商品及时学习
- 定期检查并补充样本

## API接口文档

### 自定义API对接

如需对接自有AI服务，选择"自定义API"：

#### API要求

**请求格式**：
```json
POST /api/recognize
Content-Type: application/json

{
  "image": "base64_encoded_image",
  "threshold": 0.6
}
```

**响应格式**：
```json
{
  "success": true,
  "result": {
    "productId": "PROD_001",
    "productName": "红富士苹果",
    "confidence": 0.85,
    "alternatives": [
      {
        "productId": "PROD_002",
        "productName": "阿克苏苹果",
        "confidence": 0.72
      }
    ]
  }
}
```

## 故障排查

### 问题1：识别不准确

**解决方案**：
- 检查商品学习样本数量（建议≥20张）
- 补充多角度、多状态样本
- 调整相似度阈值（提高阈值更严格）

### 问题2：无法触发识别

**解决方案**：
- 检查AI识别开关是否开启
- 检查触发重量设置
- 确认称重传感器工作正常
- 检查触发模式设置

### 问题3：摄像头无法启动

**解决方案**：
- 检查摄像头权限是否授予
- 重启APP
- 检查摄像头是否被其他应用占用

### 问题4：API调用失败

**解决方案**：
- 检查网络连接
- 验证API地址和密钥是否正确
- 查看API返回的错误信息
- 检查API响应格式是否符合要求

## 注意事项

1. **首次使用**：需要为常用商品进行学习，建议优先处理高频商品
2. **持续优化**：日常运营中遇到识别错误要及时纠正和学习
3. **数据备份**：学习样本会保存在本地，建议定期备份
4. **硬件要求**：需要200万像素以上的摄像头
5. **环境要求**：避免过暗或过亮的环境
