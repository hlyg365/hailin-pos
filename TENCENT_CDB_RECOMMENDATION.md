# 腾讯云CDB PostgreSQL 版本与配置推荐

## 🎯 版本推荐

### 推荐版本：PostgreSQL 14.x 或 16.x

| 版本 | 特性 | 推荐度 | 说明 |
|------|------|--------|------|
| **PostgreSQL 16** | 最新特性，性能优化 | ⭐⭐⭐⭐ | 长期支持，适合新项目 |
| **PostgreSQL 14** | 成熟稳定，生态完善 | ⭐⭐⭐⭐⭐ | **首选**，稳定可靠 |
| PostgreSQL 13 | 经典版本 | ⭐⭐⭐ | 兼容性好 |
| PostgreSQL 12 | 老旧版本 | ⭐⭐ | 不推荐 |

**推荐理由**：
- ✅ PostgreSQL 14 是目前最成熟稳定的版本
- ✅ PostgreSQL 16 是最新LTS版本，功能更强
- ✅ 腾讯云CDB对这两个版本支持最好
- ✅ 社区生态完善，资料丰富

## 🔧 具体配置清单

### 1. 基础配置

| 配置项 | 推荐值 | 说明 |
|--------|--------|------|
| **地域** | 广州/上海/深圳 | 选择离用户最近的 |
| **可用区** | 单可用区（入门）/ 多可用区（生产） | 高可用需要多可用区 |
| **PostgreSQL版本** | 14.5 或 16.1 | 推荐14或16 |
| **实例规格** | 参考上文场景配置 | 根据业务规模 |
| **存储空间** | 起步50GB，后续按需扩展 | 支持在线扩容 |
| **存储类型** | SSD云硬盘（默认） | 高性能需求可选增强SSD |
| **网络** | VPC私有网络 | 必须，安全隔离 |

### 2. 网络配置

```
必须配置：
✅ VPC私有网络（安全隔离）
✅ 配置子网（建议10.0.0.0/16）
✅ 配置安全组（仅开放必要端口）

安全组规则：
- 入站规则：
  • PostgreSQL：5432端口（仅允许应用服务器IP）
  • HTTPS：443端口（如需要管理界面）
- 出站规则：
  • 全部放通（或按需配置）
```

### 3. 备份配置

```
必须配置：
✅ 自动备份：每日全量备份
✅ 备份保留：7天（建议）
✅ 开启binlog：是的（用于恢复）
✅ 开启审计日志：是的（安全合规）

推荐配置：
✅ 跨地域备份：可选（重要数据建议开启）
✅ 定期演练：每月测试恢复
```

## 📝 购买后配置

### 1. 创建数据库和账号

```sql
-- 登录数据库（通过DMS或本地客户端）
-- 腾讯云控制台提供DMS数据库管理工具

-- 创建数据库
CREATE DATABASE hailin;

-- 创建用户（建议分开管理）
CREATE USER hailin_app WITH PASSWORD '你的强密码';
GRANT ALL PRIVILEGES ON DATABASE hailin TO hailin_app;

-- 创建只读账号（可选，用于报表）
CREATE USER hailin_readonly WITH PASSWORD '你的强密码';
GRANT CONNECT ON DATABASE hailin TO hailin_readonly;
GRANT USAGE ON SCHEMA public TO hailin_readonly;
GRANT SELECT ON ALL TABLES IN SCHEMA public TO hailin_readonly;
```

### 2. 配置白名单

```
腾讯云控制台 → 数据库 → 点击实例 → 找到"白名单"设置

配置项：
- 当前客户端IP自动添加
- 或手动添加IP段

示例白名单：
- 应用服务器IP：10.0.0.1/32
- 开发机器IP（仅开发阶段）：xxx.xxx.xxx.xxx/32
- 全部放通：0.0.0.0/0 ⚠️ 不推荐，仅测试用
```

### 3. 获取连接信息

```
腾讯云控制台 → 数据库 → 点击实例 → 基本信息

你会看到：
- 连接地址：your-instance.tencentcdb.com
- 内网端口：5432
- 数据库账号：hailin_app
- 初始密码：创建时设置
```

### 4. 环境变量配置

```env
# 腾讯云CDB PostgreSQL连接
DATABASE_URL=postgresql://hailin_app:密码@your-instance.tencentcdb.com:5432/hailin

DB_HOST=your-instance.tencentcdb.com
DB_PORT=5432
DB_NAME=hailin
DB_USER=hailin_app
DB_PASSWORD=你的密码

# SSL连接（腾讯云推荐开启）
DB_SSL_MODE=require
```

## 🔗 与扣子平台集成

### 架构方案

```
┌─────────────────────────────────────────────────────────────┐
│                        腾讯云                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌──────────┐              ┌─────────────────────┐       │
│   │  扣子平台  │────────────▶│   腾讯云CDB         │       │
│   │  (Next.js)│   内网访问   │  PostgreSQL 14      │       │
│   └──────────┘              └─────────────────────┘       │
│                                                             │
│   ┌──────────┐              ┌─────────────────────┐       │
│   │  小程序   │────────────▶│                     │       │
│   └──────────┘              │                     │       │
│                             │                     │       │
│   ┌──────────┐              │                     │       │
│   │  收银台   │────────────▶│                     │       │
│   └──────────┘              └─────────────────────┘       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 扣子平台连接腾讯云

**前提条件**：
1. 扣子平台和腾讯云在同一地域
2. 使用VPC私有网络连接
3. 配置内网访问

**腾讯云配置**：
```
1. 创建VPC私有网络
   - VPC CIDR：10.0.0.0/16
   - 子网：10.0.1.0/24

2. 配置云联网（可选）
   - 连接扣子平台到腾讯云VPC
   - 配置路由表

3. 配置安全组
   - 允许扣子平台IP访问5432端口
   - 限制源IP
```

### 获取连接信息

```
购买后获取：
- 内网地址：your-instance.tencentcdb.com（内网域名）
- 内网端口：5432

配置到扣子平台环境变量：
DATABASE_URL=postgresql://用户名:密码@your-instance.tencentcdb.com:5432/数据库名
```
