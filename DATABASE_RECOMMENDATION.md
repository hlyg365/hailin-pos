## 📝 购买建议

### 1. 阿里云购买步骤

```
1. 访问阿里云官网
2. 注册/登录阿里云账号
3. 进入 RDS PostgreSQL 产品页
4. 选择【立即购买】
5. 配置：
   - 地域：选择离用户最近的（如华南-广州）
   - 版本：PostgreSQL 14或16
   - 规格：入门型（1核1G）
   - 存储：50GB
   - 购买时长：1年（享折扣）
6. 确认订单并支付
7. 创建数据库和账号
8. 获取连接信息
```

### 2. 连接信息配置

购买后，你会获得：
```
- 主机地址：pgm-xxxxxxxxxxxx.pg.rds.aliyuncs.com
- 端口：5432
- 数据库名：hailin
- 用户名：hailin_user
- 密码：xxxxxx
```

### 3. 环境变量配置

```env
# 阿里云RDS连接
DATABASE_URL=postgresql://hailin_user:密码@pgm-xxxxxxxxxxxx.pg.rds.aliyuncs.com:5432/hailin

DB_HOST=pgm-xxxxxxxxxxxx.pg.rds.aliyuncs.com
DB_PORT=5432
DB_NAME=hailin
DB_USER=hailin_user
DB_PASSWORD=密码
```

## ⚠️ 注意事项

### 安全配置（必须做）

```
1. ✅ 开启SSL连接
2. ✅ 设置强密码
3. ✅ 配置IP白名单（仅允许应用服务器IP）
4. ✅ 启用VPC私有网络
5. ✅ 开启审计日志
6. ✅ 定期修改密码

```
1. ✅ 开启自动备份（每日）

3. ✅ 定期手动备份重要数据
