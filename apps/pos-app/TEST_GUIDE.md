## 测试流程

### 首次登录测试
1. 访问 http://localhost:5000/
2. 点击右上角"登录"按钮
3. 输入任意测试账号信息
4. 点击"登录"按钮
5. 验证跳转到对应页面

### 完整业务流程测试
1. **收银流程**：登录收银台 → 添加商品 → 识别会员 → 支付 → 完成
2. **要货流程**：店长助手 → 库存预警 → 要货申请 → 总部审批 → 配送
3. **缴款流程**：店长助手 → 缴款 → 提交 → 总部确认
4. **促销流程**：总部 → 创建促销 → 生效 → 门店可见 → 核销

---

## 注意事项

1. **当前为演示版本**：登录时任意输入即可通过验证
2. **数据存储在内存**：刷新页面后数据会重置
3. **晚8点清货**：系统时间20:00-23:00自动开启
4. **离线模式**：断开网络后显示离线提示
