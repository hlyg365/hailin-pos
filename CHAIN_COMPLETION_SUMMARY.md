## 📖 使用说明

### 快速开始

```bash
# 1. 安装依赖
cd hailin-ministore
pnpm install

# 2. 运行小程序
# 使用HBuilderX或微信开发者工具
# 导入 hailin-ministore 项目

# 3. 配置小程序
# 在微信开发者工具中配置AppID
# 配置服务器域名

# 4. 测试功能
# 测试所有核心功能
# 修复发现的问题
```

### 主要操作

#### 创建接龙
1. 点击"创建接龙"按钮
2. 填写接龙信息（标题、描述、价格）
3. 设置截止时间
4. 添加商品清单
5. 点击"发布接龙"

#### 参与接龙
1. 在接龙列表中选择接龙
2. 点击"我要参与"按钮
3. 确认参与信息
4. 完成支付
5. 等待接龙开始

#### 管理接龙
1. 在"我的接龙"中找到接龙
2. 点击"管理"按钮
3. 修改接龙设置
4. 管理商品和参与者
5. 查看接龙数据
